# Quant Strategies

Personal collection of algorithmic trading strategies built in Python.

## Strategy 1 — MA Crossover Backtest (SPY 2018-2026)

Built as a learning project to understand the mechanics of systematic backtesting. 

Key finding: simple MA crossover strategies are too laggy to compete with buy & hold on trending stocks. Next iteration will incorporate faster signals — RSI, volume, and sentiment analysis.

### Performance Metrics
- Cumulative Return vs Buy & Hold
- Sharpe Ratio
- Annualised Volatility
- Max Drawdown

### Tools
Python, pandas, yfinance, matplotlib, numpy
